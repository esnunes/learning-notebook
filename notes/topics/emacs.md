# Emacs

## How to install on Mac OS X

```bash
brew install emacs --with-cocoa --srgb
```

## Keyboard shortcuts

- <kbd>M-:</kbd> - Evaluate Emacs Lisp expression
- <kbd>M-;</kbd> - Comment line / block
- <kbd>C-_</kbd> - Undo
- <kbd>ESC-<</kbd> - Top of the file
- <kbd>ESC-></kbd> - Bottom of the file
